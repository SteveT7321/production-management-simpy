# C01：5S 評核工具

## 概念說明

5S 是日本精實製造的基礎管理文化，依序為：

| S | 日文 | 中文 | 核心行動 |
|---|------|------|---------|
| S1 | 整理（Seiri） | Sort | 清除不需要的物品 |
| S2 | 整頓（Seiton） | Set in Order | 每樣東西有固定位置 |
| S3 | 清掃（Seiso） | Shine | 每日清潔＋設備點檢 |
| S4 | 清潔（Seiketsu） | Standardize | SOP 文件化、標準維持 |
| S5 | 素養（Shitsuke） | Sustain | 員工自律、稽核機制 |

## 為什麼不用 SimPy？

5S 是**組織文化成熟度**的靜態評核，沒有隨機事件或時間流逝需要模擬。
但 5S 分數可以量化映射到生產效率指標，本工具提供此映射。

## 工具功能

```
python concepts/c01_5s/calculator.py
```

輸出三個表格：
1. **5S 評核結果**：各維度現況分數 vs 目標分數
2. **效率影響估算**：搜尋浪費時間、不良率、工安事故改善幅度
3. **查核點細節**：每個查核項目的現況/目標/差距

## 預設情境

- **現況（SMT 廠區）**：總分 12/60（20%），多數維度偏低
- **目標**：總分 45/60（75%），達到業界一般水準

## 5S 與效率的量化關係

| 5S 分數 | 搜尋浪費 | 備註 |
|--------|---------|------|
| < 40%  | 18% 工時 | 嚴重混亂 |
| 40–60% | 12% 工時 | 一般工廠 |
| 60–80% | 7% 工時  | 良好狀態 |
| > 80%  | 3% 工時  | 精實標竿 |

資料來源：SME / Lean Enterprise Institute 工廠改善案例彙整

## 與其他章節的連結

```
5S（本工具）
  └─ S2 整頓 → 視覺化管理 → Kanban（Ch06）
  └─ S3 清掃 → 設備點檢 → PM（Ch07）
  └─ S4 清潔 → SOP 標準化 → SPC（Ch11）
```

5S 是所有精實工具的地基，分數不足時其他工具效果會大打折扣。
